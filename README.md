# 💊 Health & Schedule Reminder – VS Code Extension

Stay healthy while coding! Get reminders for tea breaks, water, stretching, eye rest, and meetings — all with a music alarm popup!

## ✨ Features
- ☕ Tea Break reminder (every 60 mins)
- 💧 Water reminder (every 30 mins)
- 🧘 Stretch reminder (every 90 mins)
- 👁️ Eye rest 20-20-20 rule (every 20 mins)
- 📅 Custom meeting alerts with 5-min pre-alert
- 🔔 Music alarm beep on every popup
- 📊 Status dashboard

## 🚀 Commands
- `Start Health Reminders`
- `Stop Health Reminders`
- `Add Meeting Alert`
- `Show Reminder Status`

## ⚙️ Settings
- `healthReminder.teaBreakInterval` – default 60 mins
- `healthReminder.waterInterval` – default 30 mins
- `healthReminder.stretchInterval` – default 90 mins

## 📦 Install
1. Download the `.vsix` file
2. Open VS Code → Extensions → `...` → Install from VSIX
